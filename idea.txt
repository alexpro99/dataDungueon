derechos del sistema:
en el servidor hay una api con endpoints que son las habilidades 
estas piden un nivel especifico
como si fueran roles y permisos

habilidades: {
    ej:{
        ataque con arma {
            realiza un calculo con las stats del player para sacar un daño y eso le mete al monstruo
        }
    }
    tipos: {
        pasivas
        activas: {
            daño,
            buffos
        }
        entorno: {
            son generales.. mover(gasta stamina), descansar(para recuperar stamina y mana)
        }

    }

}

sistema de combate {
    pvp: {
        
    }

    pve: {
        hay laberintos que son matrices o estructuras de datos que contienen recompenzas y enemigos
        cuando el jugador entra al laberinto se crea una instancia del mismo en la bd relacionado con el player
        se le manda una copia de la matriz al cliente con una informacion incompleta o completa segun sensorial(hab)
        que es la 'vision' del player. 
        -el jugador debe recorrer la matriz hasta llegar al nodo salida, si lo logra se le da una recompenza 
        -si hay enemigos debe usar sus habilidades para acabar con ellos
        -los enemigos atacan despues de recibir, esquivar o bloquear la habilidad del jugador
        -al morir los enemigos sueltan una recompensa
        -mana y estamina se gasta segun las habilidades
                

    }
}
